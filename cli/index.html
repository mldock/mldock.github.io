# Command Line Tool

Core to MLDOCK is a command line tool which streamlines the development of machine learning containers by leveraging the docker tool stack and sprinkling in some opinionated configuration to empower you.

To deliver upon this, MLDOCK provides a set of command groups:

Command Group Description
container A set of commands that support creating new containers, initialize and update containers.
local A set of commands to use during the development phase. Creating your trainer, prediction scripts and debugging the execution of scripts.
registry A set of tools to help you push, pull and interact with image registries.
configure A collection of commands to set up your CLI configuration.
